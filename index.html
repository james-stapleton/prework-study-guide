HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Hypertext markup language
  • This is a list
  • This list is in a section
  • This section is in the body
  • The user will see all of this
  • I can make headings, sub headings, lists, add an img, and new sections
  • Soon I'll learn more

CSS

  • Cascading Style Sheet
  • This page is formatted with an external style sheet
  • External sheet must be linked in head
  • sections must be assigned a class and the class must be specified in the style sheet
  • CSS formats borders, padding, margins, font, colors, sizes and more

Git

  • all commands start with git
  • git checkout -b (branch-name): create and switch to new branch
  • git status: current branch
  • -A adds all changes
  • commit -m "" adds message with commit
  • git pull origin main specifies remote base branch as source

JavaScript

  • JS is a dynamic, loosely typed language
  • uses variables, functions, conditional statements, arrays, etc.
  • Brings interactivity to a webpage
  • Is linked in the body, unlike CSS external sheets which are linked in head
  • Can be used to detect events like a mouse click or to generate a random output
